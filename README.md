📊 Superstore Sales Dashboard

📌 Project Overview
This project focuses on Data Cleaning, Visualization, and Dashboard Creation using Python and Power BI.

The goal of this project is to transform raw sales data into meaningful business insights through data preprocessing, analysis, and interactive visual dashboards.

🚀 Features
- Data Cleaning and Preprocessing
- Handling Missing Values and Duplicates
- Outlier Detection
- Data Visualization using Charts
- Interactive Dashboard using Power BI
- Business Insights and Trend Analysis

 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

📊 Dashboard Components
- KPI Card (Total Sales)
- Regional Sales Analysis
- Category-wise Sales Comparison
- Sales Trend Over Time
- Interactive Filters (Slicers)
- Detailed Data Table
  
📈 Key Insights
- Technology category generated the highest sales
- West region showed strong performance
- Sales increased significantly over time
- Interactive filters improve analysis efficiency

 📂 Project Files
- `dashboard.pbix` → Power BI Dashboard
- `cleaned_superstore.csv` → Cleaned Dataset
- `analysis.ipynb` → Python Analysis Code
- `dashboard.png` → Dashboard Screenshot
